# AGENTS.md

## Core workflow
1. Read the existing game package if present
2. Classify the request as create, expand, revise, or repair
3. Update the smallest correct set of files
4. Validate the package before finishing
5. Return a compact summary of changes

## Rules
- Prefer templates over freeform prose
- Prefer one file per scene and one file per character
- Keep hidden secrets explicit in canon files
- Keep authoring and runtime concerns separate
